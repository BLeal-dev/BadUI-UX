# BadUI-UX
📘 Projeto – Psicologia 01

Este projeto foi desenvolvido como parte da disciplina Psicologia 01, do curso de Licenciatura em Ciência da Computação da Universidade Federal Rural de Pernambuco (UFRPE).

🎯 Objetivo

O objetivo da atividade foi criar ou demonstrar um exemplo de má experiência do usuário (UX) em um sistema digital, aplicando conceitos estudados ao longo da disciplina.

💡 Proposta do Grupo

Nosso grupo decidiu desenvolver um sistema simples que evidencia, na prática, situações de usabilidade ruim, com foco em:

Interfaces confusas

Navegação pouco intuitiva

Feedback inadequado ao usuário

Problemas de acessibilidade e interação

A ideia é permitir que o usuário vivencie diretamente uma experiência negativa, facilitando a análise crítica dos erros de design.

🛠️ Tecnologias Utilizadas

• HTML

• CSS

• JavaScript

📚 Contexto Acadêmico

Este projeto busca conectar conceitos de Psicologia com o desenvolvimento de software, mostrando como fatores humanos influenciam diretamente na qualidade da experiência do usuário.
